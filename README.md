# Raffles App

## English

A simple web app for randomly drawing teams from a list of participants.

### Features

- Add any number of participants, one per line
- Choose how many teams to split them into (2 to 6)
- Give the draw a custom title
- Teams are shuffled using the Fisher-Yates algorithm and distributed evenly via round-robin
- Results are displayed in a responsive card grid with a player count per team
- Inline validation errors on each field — no alert() dialogs

### Tech stack

Plain HTML, CSS and JavaScript — no frameworks, no dependencies.

## Português

Aplicação web simples para sortear equipes aleatórias a partir de uma lista de participantes.

### Funcionalidades

- Adicione qualquer número de participantes, um por linha
- Escolha em quantas equipes dividir (2 a 6)
- Dê um título personalizado ao sorteio
- Os participantes são embaralhados com o algoritmo de Fisher-Yates e distribuídos de forma equilibrada via round-robin
- Resultado exibido em um grid responsivo de cards com contagem de jogadores por equipe
- Erros de validação inline em cada campo — sem caixas de alerta

### Tecnologias

HTML, CSS e JavaScript puro — sem frameworks, sem dependências externas.
